# 📈 Stock Portfolio Tracker

## Description

This project is developed as part of the CodeAlpha Python Programming Internship.

The application allows users to enter stock names and quantities, calculates the total investment value using predefined stock prices, and saves the result in a text file.

## Features

- User-friendly console interface
- Calculates total investment value
- Uses predefined stock prices
- Saves output to `portfolio.txt`

## Technologies Used

- Python 3.12

## Project Structure

```
CodeAlpha_Task2_StockPortfolio/
│── Task2_StockPortfolio.py
│── portfolio.txt
│── README.md
```

## Running Locally

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python Task2_StockPortfolio.py
```

## Output

The program displays the total investment value and stores it in `portfolio.txt`.
